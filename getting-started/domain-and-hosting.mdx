域名和主机是你整个网站的地基。域名决定了用户如何找到你，主机决定了你的内容存放在哪里、加载有多快。很多新手在这一步要么花了冤枉钱，要么选了一个后期很难更换的方案。这篇文章帮你理清思路，买之前先想清楚，避开常见的坑。

## 域名选择原则

好域名的核心标准只有一个：让别人第一次听到就能记住并拼写出来。围绕这个目标，有以下几点可以参考：

<CardGroup cols={2}>
  <Card title="✅ 好域名的特征" icon="circle-check">
    - 长度在 6–14 个字符之间
    - 使用 `.com` 后缀（全球认知度最高）
    - 与品牌或主题直接相关
    - 纯字母，无连字符、无数字替代
    - 朗读出来和拼写一致
  </Card>
  <Card title="❌ 尽量避免的做法" icon="circle-xmark">
    - 超过 20 个字符的长域名
    - 使用连字符（如 `my-site.com`）
    - 用数字替代字母（如 `w3b.com`）
    - 侵犯已有品牌的商标
    - 仅因便宜而选择冷门后缀
  </Card>
</CardGroup>

<Note>
`.com` 并非唯一选择，但在国内外用户心智中仍然是"正规网站"的代名词。如果 `.com` 已被注册，优先考虑重新取名，而不是退而求其次选 `.net` 或 `.info`。面向国内用户的站点，`.cn` 也是可接受的选项。
</Note>

## 域名注册商对比

注册商的选择直接影响你的续费成本和管理体验。以下是目前口碑较好的几家：

<Tabs>
  <Tab title="Namecheap">
    **适合人群**：个人站长、预算有限的新手

    - 首年价格透明，`.com` 通常在 $8–$11/年
    - 免费提供 WHOIS 隐私保护（很多注册商额外收费）
    - 界面友好，DNS 管理简单直观
    - 续费价格与首年差距不大，无明显涨价陷阱

    **不足**：客服响应速度一般，遇到复杂问题需要等待。
  </Tab>
  <Tab title="Cloudflare Registrar">
    **适合人群**：已在用 Cloudflare 做 CDN 或 DNS 的站长

    - 以**成本价**出售域名，没有溢价，续费价格与首年完全一致
    - `.com` 约 $9.15/年（Cloudflare 自身的批发成本）
    - 强制绑定 Cloudflare DNS，迁移有一定门槛
    - 不支持首次注册，只能转入已有域名

    **不足**：不能从 Cloudflare 直接注册新域名，需先在其他注册商购买再转入。
  </Tab>
  <Tab title="阿里云万网">
    **适合人群**：网站主要面向中国大陆用户，需要 ICP 备案

    - 国内最大的域名注册商，备案流程顺畅
    - `.com` 首年活动价常见 ¥55 左右，续费约 ¥85/年
    - 与阿里云主机、CDN 生态打通，管理集中
    - 控制台功能完整，适合不熟悉英文界面的用户

    **不足**：价格略高于国际注册商，部分促销活动有隐性条件。
  </Tab>
</Tabs>

<Tip>
购买域名时，**一次性续费 3–5 年**通常比每年单独续费更划算，同时也能向搜索引擎传递"这个站点会长期运营"的信号，对 SEO 略有好处。
</Tip>

## 域名购买流程

<Steps>
  <Step title="头脑风暴备选名单">
    列出 5–10 个候选域名，按优先级排序。同时准备好备用方案，避免心仪的域名已被注册时措手不及。
  </Step>
  <Step title="查询可用性">
    在注册商的搜索框输入候选域名，确认可注册状态。顺便检查同名的社交媒体账号是否可注册，保持品牌统一性。
  </Step>
  <Step title="核实续费价格">
    在加入购物车之前，一定要查清**续费价格**（Renewal Price），而不只看首年价。两者差距超过 50% 的注册商要谨慎。
  </Step>
  <Step title="开启双重验证">
    完成注册后，立即在账号设置中开启双因素认证（2FA）。域名一旦被盗转，追回极其麻烦。
  </Step>
  <Step title="关闭自动续费或确认已开启">
    根据你的习惯决定：如果你容易忘记，开启自动续费；如果担心被扣款，关闭并设置日历提醒提前 60 天续费。
  </Step>
  <Step title="开启 WHOIS 隐私保护">
    大多数注册商提供免费的隐私保护，隐藏你的姓名和联系方式，防止垃圾邮件和骚扰电话。
  </Step>
</Steps>

## 主机类型对比

主机的选择需要结合你的**技术能力**、**预期流量**和**预算**综合判断。

<Tabs>
  <Tab title="共享主机">
    **原理**：多个网站共用同一台服务器的资源（CPU、内存、带宽）。

    **适合场景**：
    - 刚开始建站，日均 PV 在 1000 以下
    - 不想处理服务器运维问题
    - 预算极其有限（月费 $3–$10）

    **代表服务商**：SiteGround、HostGator、Bluehost

    **注意**：共享主机的"邻居"若流量突增，会影响你的网站速度。避免选择过于廉价（月费低于 $2）的主机，稳定性难以保证。
  </Tab>
  <Tab title="VPS 主机">
    **原理**：虚拟专用服务器，通过虚拟化技术在物理服务器上划分出独立资源，不与他人共享。

    **适合场景**：
    - 日均 PV 在 1000–50000 之间
    - 具备基本的 Linux 命令行操作能力
    - 需要安装自定义软件或特殊配置

    **代表服务商**：DigitalOcean（Droplets）、Vultr、Linode、腾讯云轻量应用服务器

    **月费参考**：$6–$40，取决于配置。
  </Tab>
  <Tab title="云主机（弹性伸缩）">
    **原理**：按需分配资源，流量高峰自动扩容，流量低谷缩减，按实际用量计费。

    **适合场景**：
    - 流量波动较大（如营销活动期间流量暴增）
    - 对稳定性和 SLA 要求高
    - 团队有一定的云服务运维能力

    **代表服务商**：AWS（EC2/Lightsail）、阿里云 ECS、腾讯云 CVM、Google Cloud

    **费用**：弹性计费，初期可能比 VPS 贵，但大流量下性价比更高。
  </Tab>
  <Tab title="托管 WordPress">
    **原理**：专为 WordPress 优化的主机环境，服务商负责更新、备份、安全加固，你只需专注写内容。

    **适合场景**：
    - 使用 WordPress 建站
    - 不想花时间处理技术运维
    - 愿意为便捷性支付溢价

    **代表服务商**：WP Engine、Kinsta、Flywheel

    **月费参考**：$25–$100+，比普通主机贵，但省去大量运维时间。
  </Tab>
</Tabs>

## DNS 配置基础

购买域名和主机后，你需要通过 DNS 将两者连接起来。以下是最常用的三个概念：

<Accordion title="A 记录（Address Record）">
将域名指向一个**IPv4 地址**。这是最基础的记录类型。

**示例**：将 `yourdomain.com` 指向主机 IP `185.199.108.153`

```
类型    主机名    值               TTL
A       @         185.199.108.153  3600
A       www       185.199.108.153  3600
```

`@` 代表根域名（即 `yourdomain.com` 本身），`www` 代表 `www.yourdomain.com`。
</Accordion>

<Accordion title="CNAME 记录（Canonical Name）">
将一个域名指向**另一个域名**（而非 IP 地址）。常用于将 `www` 子域名指向主域名，或将子域名指向第三方服务。

**示例**：将 `blog.yourdomain.com` 指向 `yourdomain.ghost.io`

```
类型     主机名    值                    TTL
CNAME    blog      yourdomain.ghost.io   3600
```

注意：根域名（`@`）不能设置 CNAME，只能用 A 记录或部分服务商提供的 ALIAS/ANAME 记录。
</Accordion>

<Accordion title="TTL（Time to Live）">
TTL 是 DNS 缓存的存活时间，单位为秒。TTL 设置得越低，DNS 变更传播越快；设置越高，全球缓存越稳定。

- **迁移前**：将 TTL 提前降低到 300（5 分钟），这样切换主机后变更能快速生效
- **稳定运行后**：可设置为 3600（1 小时）或 86400（24 小时）减少查询压力
- **DNS 全球传播**通常需要 24–48 小时，即使 TTL 很低也可能有延迟
</Accordion>

## 常见坑与注意事项

<Warning>
**续费价格暴涨陷阱**：部分主机商（尤其是 Hostgator、Bluehost）以极低的首年价格（$1.99/月）吸引注册，但续费价格可能高达 $12–$15/月。购买前务必确认续费价格，并计算 3 年总成本再做决定。
</Warning>

<Warning>
**国内备案要求**：如果你的网站使用中国大陆的服务器（IP 归属大陆），必须完成 **ICP 备案**，否则网站可能被运营商屏蔽。备案需要提供域名证书、服务器信息、负责人身份证，流程约需 7–20 个工作日。使用香港或海外服务器则无需备案，但访问速度可能稍慢。
</Warning>

<Note>
**域名和主机是可以分开购买的**：你的域名注册商和主机服务商不需要是同一家公司。很多有经验的站长会在 Cloudflare 管理 DNS，在 Namecheap 注册域名，在 DigitalOcean 托管服务器。分开管理灵活度更高，但需要自己配置 DNS。
</Note>

<Tip>
新手建议从**共享主机 + .com 域名**起步，熟悉建站流程后再考虑迁移到 VPS 或云主机。迁移网站比你想象的要容易，不必在第一步就追求最完美的配置。
</Tip>
